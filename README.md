# FitTrack

> Персональный трекер трансформации тела — полностью локальный, без облака, без подписок.

```
Питание · Тренировки · Замеры тела · Статистика · Прогноз
```

---

## Что это

FitTrack — одностраничное приложение для отслеживания физической формы. Данные хранятся у вас на диске в JSON-файлах. Никакой регистрации, никакого интернета, никакой утечки данных.

**Модули:**

| Модуль | Что делает |
|---|---|
| **Тело** | Дневник замеров: вес, % жира, 7 обхватов. Дельты, состав тела, история. |
| **Питание** | Журнал КБЖУ по приёмам пищи. Поиск через Open Food Facts. Трекер воды. |
| **Тренировки** | Запись сетов, таймер, 1RM (Brzycki), тоннаж, история по упражнениям. |
| **Дашборд** | 8 виджетов: калории, макросы, спарклайн веса, прогноз цели, стрик. |
| **Статистика** | Графики за 7/30/90 дней. Recharts. Экспорт всех данных в JSON. |

## Быстрый старт

```bash
# 1. Клонировать
git clone <repo-url>
cd FitTrackv-1.0

# 2. Установить зависимости + настроить хук
./setup.sh

# 3. Запустить
./start.sh
```

Откройте **http://localhost:5173**

> Сервер API поднимается на порту **3001**. Клиент проксирует `/api/*` автоматически.

## Формулы

| Показатель | Метод |
|---|---|
| BMR | Миффлин-Сан Жеор |
| TDEE | BMR × коэффициент активности (5 уровней) |
| Целевые ккал | TDEE ± дефицит/профицит по режиму и интенсивности |
| Рекомпозиция | +100 ккал в дни тренировок / −200 в дни отдыха |
| Белок | 2.2 г/кг сухой массы (cutting/bulking), 2.6 г (recomposition) |
| 1RM | Формула Brzycki |
| Тоннаж | Сумма (вес × повторы) по всем сетам |

## Безопасность

- Все `/api/*` маршруты доступны **только с localhost** — внешние запросы получают `403`
- Входящие данные валидируются Zod-схемами
- `data/` в `.gitignore` и защищён pre-commit хуком — случайно не закоммитить
- Helmet и CORS подключены глобально
